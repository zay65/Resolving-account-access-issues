<h2> Password Resets in MS 365 </h2>


<div align="center"> Within "Microsoft 365 admin center", select the drop down arrow of the "Users" category.
Next, select the "Active users" sub-category to view the list of active users within tenant. Then, select one or more names of the users who's password(s) you wish to reset.</div>

  </p>

  <div align="center"> Select "Reset password" twice when prompted.</div>

<h2> Self-Service Password Reset (SSPR) config </h2>

 <div align="center"> In "Microsoft Entra admin center", select the drop down arrow of the "Users" category.</div>

 <p align="center">
    <img src="https://github.com/zay65/Resolving-account-access-issues/blob/d2e3e1024f23cc924159301f684e50f69abe6e66/SSPR%20ENTRA%20ID%20MS365%201.png" alt="Sample Image"/>
  </p>



 <div align="center"> Enter the name of the group you wish to create in the text dialogue box and select "No users selected".</div>

 <p align="center">
    <img src="https://github.com/zay65/Resolving-account-access-issues/blob/d2e3e1024f23cc924159301f684e50f69abe6e66/SSPR%20ENTRA%20ID%20MS365%202.png" alt="Sample Image"/>
  </p>



 <div align="center">  Check the boxes of the users you wish to enroll in Entra ID's SSPR tool and select the "Select" button .</div>

 <p align="center">
    <img src="https://github.com/zay65/Resolving-account-access-issues/blob/d2e3e1024f23cc924159301f684e50f69abe6e66/SSPR%20ENTRA%20ID%203.png" alt="Sample Image"/>
  </p>




 <div align="center"> Select "Enable".</div>

 <p align="center">
    <img src="https://github.com/zay65/Resolving-account-access-issues/blob/d2e3e1024f23cc924159301f684e50f69abe6e66/SSPR%20ENTRA%20ID%204.png" alt="Sample Image"/>
  </p>




 <div align="center"> The group of users you chose has now been created, Entra ID Premium licenses have been assigned, and they are now enrolled and able to use SSPR via email/phone call/sms .</div>

 <p align="center">
    <img src="https://github.com/zay65/Resolving-account-access-issues/blob/d2e3e1024f23cc924159301f684e50f69abe6e66/SSPR%20ENTRA%20ID%205.png" alt="Sample Image"/>
  </p>




 <div align="center"> If specific and/or advanced settings need to be configured, select the green "success" text box above for more options..</div>

 <p align="center">
    <img src="https://github.com/zay65/Resolving-account-access-issues/blob/d2e3e1024f23cc924159301f684e50f69abe6e66/SSPR%20ENTRA%20ID%206.png" alt="Sample Image"/>
  </p>
