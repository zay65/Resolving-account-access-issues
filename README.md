<h2> Password Resets in MS 365 </h2>


<div align="center"> Within MS 365 admin center, select the drop down arrow of the "Users" category.
Next, select the "Active users" sub-category to view the list of active users within tenant. Then, select one or more names of the users who's password(s) you wish to reset.</div>
